MCP

שרתי MCP — הרחבות ל-Claude

Model Context Protocol (MCP) הוא תקן פתוח שמאפשר ל-Claude להתחבר לכלים חיצוניים — בסיסי נתונים, APIs, דפדפנים ועוד. כל MCP הוא שרת קטן שרץ מקומית ומוסיף יכולות חדשות.

⏱ 10 דקות קריאה רמה: מתחיל

מה זה MCP ולמה זה חשוב

ברירת המחדל של Claude Code היא לעבוד עם קבצים וטרמינל. MCP מרחיב את זה — Claude יכול לשאול את ה-MCP שאלות ולקבל מידע אמיתי בזמן אמת: לפתוח PR ב-GitHub, לבצע query ב-Postgres, לבדוק תשלום ב-Stripe.

MCP שרתים מוגדרים בקובץ ~/.claude/claude_desktop_config.json (בגירסה הדסקטופ) או ב-claude_code_config.json בפרויקט.

הוספת MCP שרת

יש שתי דרכים להוסיף MCP: דרך פקודת claude mcp add (הדרך המהירה), או ידנית בקובץ ה-JSON.

  1. הוספה מהירה עם CLI

    הפקודה הבאה מוסיפה MCP ומגדירה אותו אוטומטית:

    claude mcp add github -- npx -y @modelcontextprotocol/server-github
  2. הגדרה ידנית ב-JSON

    פותחים ~/.claude/claude_desktop_config.json ומוסיפים: 

    {
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "ghp_..."
      }
    }
  }
}
  3. אימות ההתקנה

    בתוך Claude, בודקים שה-MCP פעיל:

    /mcp
אבטחה לעולם אל תשמרו API keys ישירות בקוד. השתמשו במשתני סביבה (env) בתוך הגדרת ה-MCP, או בכלי ניהול סודות כמו direnv.

6 שרתי MCP חיוניים

🐙
GitHub
פתיחת PR, קריאת issues, חיפוש קוד, ניהול repos — ישירות מהשיחה עם Claude.
🐘
PostgreSQL
ביצוע queries, בדיקת schema, ניתוח נתונים — בלי לצאת מהשיחה.
📚
Context7
גישה לתיעוד עדכני של ספריות (React, FastAPI, Next.js ועוד) בזמן כתיבת קוד.
🎭
Playwright
אוטומציה של דפדפן — Claude יכול לנווט, ללחוץ, ולצלם screenshot בזמן אמת.
💳
Stripe
בדיקת תשלומים, ניהול customers, פתרון בעיות webhook — ישירות מהשיחה.
⚡
Supabase
ניהול בסיס נתונים, auth, ו-storage ב-Supabase — ללא כניסה ל-dashboard.

דוגמת שימוש — GitHub MCP

לאחר הגדרת ה-MCP, ניתן לבקש מ-Claude לפתוח PR, לבדוק issues פתוחים, או לחפש קוד ב-repo:

שיחה עם claude 
open a PR for the current branch against main.
title: "feat: add /health endpoint"
include a summary of the changes in the PR body

Claude ישתמש ב-GitHub MCP כדי לקרוא את ה-diff, לכתוב תיאור, וליצור את ה-PR — הכל בלחיצה אחת.

טיפ ניתן להגדיר MCP שרתים ברמת פרויקט (בתיקיית .claude/) ולא רק גלובלית. כך כל פרויקט יכול להשתמש ב-MCP שונים בהתאם לצרכיו.
